受限于答辩现场无法长时间安全运行高压/复杂实验设备，我们特此开发了这套**前端数字孪生沙盒页面**。本页面在本地浏览器中完整封装并运行了真实的 `TensorFlow.js` 深度学习环境，通过加速的数学仿真模型，直观、可视化地向各位评审老师呈现我们的算法核心与系统演进过程。

---

### 一、 核心界面与功能模块剖析

本控制台以“数字孪生”的理念划分为四大功能面板，展示了从“环境感知”到“算法决策”，再到“硬件执行”的完整闭环。

#### 1. 顶部状态栏与核心中枢 (Header)
*   **状态监测**：实时显示 MCU 控制器的在线/离线与数据同步状态。
*   **开始学习 (Start Learning) 按钮**：**系统的核心驱动键**。点击后，沙盒将切换至“全自动闭环寻优模式”，锁定人工干预。底层 AI 会正式接管硬件控制权，基于当前环境数据展开自主试错与参数搜索，以求达到项目的最优操作解。

#### 2. 环境传感感知矩阵 (Sensors Panel)
位于界面左侧，模拟了工业现场的四个核心物理化学指标的实时数显与回溯曲线。
*   **蓝藻密度 (Algae)**、**酸碱度 (pH)**、**水体温度 (Temp)**、**反应电导率 (Cond)**。
*   **功能说明**：在实际系统中，这些由实地传感器传回的数据构成了环境状态向量，是 AI 评判当前处理效果并且制定下一步输出策略的“环境观测值”。

#### 3. 决策拓扑与性能观测核心 (BO-DNN Core)
位于界面正中央，是向各位老师展示“AI 究竟在想什么、做得怎么样”的透明化窗口。
*   **能耗与灭活态势博弈图 (Evaluation Topology)**：展示了灭活率提升与物理能耗增加之间的拉锯过程。算法的终极目标就是在这张双 Y 轴图表上寻找并稳定在“帕累托最优点”。
*   **算法诊断：代理网络损失 (DNN Surrogate Loss)**：展示后台深度神经网络对复杂生化及物理反应的“摸索与拟合”程度。数值越低，说明神经网络构建的虚拟物理模型越逼近真实世界。
*   **算法诊断：UCB置信界探索 (UCB Bounds Exploration)**：贝叶斯优化的核心机制。散点分布展现了系统在“探索未知更优参数（Exploration）”与“压榨已知最优参数（Exploitation）”之间的数学权衡。
> **💡 特别说明：为何图表刷新如此之快？**
> 在真实的工业场景中，化学/物理反应的延时意味着完成一次“采样-决策”迭代可能长达数十分钟。**为了结项演示的直观性，我们将原本缓慢的物理时间轴在沙盒中压缩加速到了每秒 1 个 Tick。** 老师们看到的每秒图形抽搐与跳动，实际上是底层的张量推演在进行“高频超算展示”。

#### 4. 硬件链路与系统日志 (Hardware & Terminal)
*   **控制参数链路 (Hardware)**：包括“球隙激发距离”、“钨针辐射间距”、“电容脉冲能级”三个硬件调节滑块。在非 AI 状态下，老师可随意拉动滑块，感受手动调节参数带来的水质与能耗变化；在 AI 接管后，该面板受算法直接调控。
*   **系统日志 (System Logs)**：详实打印了系统底层的每一项网络超参数更新与伺服变动，供后续保存与数据溯源。

### 二、 实机演进计划：从“沙盒”走向“现场” (Next Steps)

当前版本圆满达成了大创项目“跑通软硬件联调协议与 AI 验证闭环”的阶段性目标。下一步的研发核心将聚焦于从“全数字孪生仿真”向“实体物理交火”的转化：

1.  **对接实体传感器阵列**：
    剥离当前前端预设的反馈模拟公式，将前端通过 `WebSocket` 或 `MQTT` 协议直连树莓派或工业 PLC。从真实的水质探头中提取真实的 Algae、pH 与温度数据流。
2.  **驱动伺服步进电机**：
    AI 运算得出的“极距”、“球隙”等硬件参数指令，将直接下发至下位机（如 Arduino/ESP32 控制板），由步进电机驱动丝杠，实现高压放电间距的真实物理推拉响应。
3.  **计算资源云端化**：
    将浏览器上运行的 `TF.js` 轻量级计算任务，迁移至实验室内的高性能 Python (PyTorch) 运算服务器，以应对未来现场高频、多维并发的工业数据集。

### 三、 本地部署与运行指南 (Deployment Guide)

如果老师或评审组成员希望在自己的 Mac 或 Windows 电脑上本地运行并查阅本页面（从 GitHub 下载了项目压缩包），只需极简的几步即可启动本环境。

#### 1. 环境准备
请确保您的电脑上已安装 **Node.js**（推荐 v18 或更高版本）。您可以在 [Node.js 官网](https://nodejs.org/) 下载并默认安装。

#### 2. 启动步骤
1. **解压文件**：将您从 GitHub 下载的 `.zip` 文件解压到一个文件夹中。
2. **打开终端**：
   - **Win 用户**：在解压后的文件夹空白处，右键选择“在终端中打开 (Open in Terminal)”，或打开 CMD并通过 `cd` 命令进入该目录。
   - **Mac 用户**：打开“终端 (Terminal)”，将解压后的文件夹拖入终端窗口前输入 `cd `（注意尾部有空格），进入该目录。
3. **安装系统依赖**：
   在终端中输入以下命令并回车，等待依赖包拉取完成（需要联网）。
   ```bash
   npm install
   ```
4. **编译并启动数字沙盒**：
   依赖安装完成后，输入以下命令启动本地服务：
   ```bash
   npm run dev
   ```
5. **在浏览器中查看**：
   终端中会输出一个本地网络地址（通常是 `http://localhost:3000` 或 `http://localhost:5173` ）。
   按住 `Ctrl` (Win) / `Cmd` (Mac) 键单击该链接，或将其复制到 Chrome/Edge 浏览器中，即可体验并检阅完整的数字孪生自动寻优控制台。
